# PREDIKSI PENJUALAN SUKU CADANG PADA PITCAR SERVICE MENGGUNAKAN ALGORITMA RANDOM FOREST 
Penelitian ini mengembangkan model prediksi penjualan suku cadang di Pitcar Service menggunakan algoritma Random Forest. Model menunjukkan performa yang baik berdasarkan metrik MAE, RMSE, dan MAPE, serta mampu menangkap pola non-linear dalam data. Hasil prediksi divisualisasikan melalui dashboard untuk membantu pengambilan keputusan strategis. Meskipun demikian, terdapat keterbatasan pada dataset dan belum mempertimbangkan faktor eksternal seperti musim dan promosi. Secara keseluruhan, Random Forest terbukti efektif sebagai pendekatan prediksi penjualan berbasis data.
